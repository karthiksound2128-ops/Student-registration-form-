# Student-registration-form-
🎓 Student Performance Analytics System
A web-based student performance analysis tool built using HTML, CSS, and JavaScript.

This project helps teachers and educators analyze student performance quickly by calculating percentages, determining pass/fail status, and generating overall class reports with a single click.

🚀 Demo
https://deepgit-hub.github.io/JavaScript-Mini-Project-5/
🚀 Features
Add student details dynamically

Enter marks for 5 subjects

Automatic percentage calculation

Automatic Pass/Fail evaluation

Department and Year management

Generate class performance reports

Calculate:

Total Students
Passed Students
Failed Students
Average Percentage
Highest Percentage
Lowest Percentage
Input validation to prevent empty submissions

Responsive user interface

🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
📊 How It Works
Enter student details:

Name
Five Subject Marks
Department
Year
Click Add Student

The system automatically:

Calculates the student's percentage
Determines Pass/Fail status
Stores student data in the table
Click Generate Report

The system generates:

Total Students
Passed Students
Failed Students
Average Percentage
Highest Percentage
Lowest Percentage
